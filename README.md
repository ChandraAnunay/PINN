# Physics-Informed Neural Networks (PINNs)

This repository contains a collection of Physics-Informed Neural Network (PINN) implementations developed while learning scientific machine learning and automatic differentiation with PyTorch.

The notebooks explore how ordinary differential equations, eigenvalue problems, and simple physics simulations can be solved by incorporating physical constraints directly into the loss function.

## Repository Structure

### ODEs
- Automatic differentiation with PyTorch
- Exponential growth equation
- Polynomial differential equations
- Trigonometric solutions
- Harmonic oscillator
- Non-linear ODEs

### Eigenvalue Problems
- Infinite square well using PINNs
- Simultaneous learning of eigenfunctions and eigenvalues

### Molecular Dynamics
- Notes and theory
- Lennard-Jones molecular dynamics simulation
- Neural network approximation of the Lennard-Jones potential

### PDEs
- Automatic differentiation exercises for PDE applications

## Tools

- Python
- PyTorch
- NumPy
- Matplotlib

This project was created as a hands-on exploration of physics-informed machine learning, automatic differentiation, differential equation solvers, and scientific computing.
